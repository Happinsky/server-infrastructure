#Infrastruktura

Repozytorium dokumentuje konfigurację i zarządzanie prywatnym serwerem VPS oraz infrastrukturą sieciową.

### Kluczowe informacje:
* **Zarządzanie Linuxem:** Obsługa serwera (Debian/Ubuntu) w pełni z poziomu terminala.
* **Zasoby systemowe:** Bieżąca kontrola zużycia procesora, pamięci RAM oraz miejsca na dysku (widoczne na zrzutach ekranu).
* **Automatyzacja:** Wykorzystanie skryptów powłoki (.sh) do zarządzania procesami i usługami systemowymi.
* **Bezpieczeństwo:** Dostęp zabezpieczony kluczem wygenerowanym w programie PuTTYgen. Logowanie tradycyjnym hasłem zostało wyłączone.
* **Sieć (Cloudflare):** Obsługa domeny, rekordów DNS (A, CNAME, SRV) oraz ochrona serwera przez Cloudflare Proxy.
